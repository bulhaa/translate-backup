The main screen

In the main screen users can enter English text and translate it to Dhivehi. If the translation is successful the Dhivehi translation is displayed. If a matching rule is not found a word-by-word translation is displayed. And if the definition is missing then the English word itself it displayed.

In all pages you can find the navigation menu and the footer. From the navigation menu you can navigate to the Main Page, Add Definition and Rule page and Download Dictionary page. In the footer some information about the project and the developers is present. A link is present to contact the team by email. The footer also contains credits for Apache OpenNLP and Materialise.

Where to go to add/edit/remove a record?

Users can add Definitions and Rules in the Add Definition and Rule page. The icon with the plus sign will take you to this page. You can enter a sentences and click Lookup. It will show the current definitions if present. The Dhivehi text is highlighted in green if this is the case. Otherwise the Dhivehi text is highlighted in red. Even if the definition is unavailable the word type will be found using Apache OpenNLP. You will need to fill the missing Dhivehi text and click Add. This will show a feedback informing whether the definition was added successfully. To add a rule you can find the form below. You can rearrange or even remove words to form the rule. Click Add rule when you are done. A feedback will be displayed below informing whether the rule was added successfully.

When adding definitions if a definition with the same English text and word type exists then, the new definition will not overwrite the existing definition. The new definition will be stored as reference but will not be used in translation. The definition correction will be reviewed by the moderator. For rules also if a clashing rule with higher precedence already exists the new rule will be added but it will be ignored. Please email us via the link in the Developers section if a rule needs to be changed.

How to download dictionary?

You can click the Download icon in the navigation menu to go to download page. This page has an icon with a down arrow. In this page you can click to download dictionary. The page also has a link which will guide you on how to import the downloaded CSV file into Microsoft Excel. You can also download the list of word types. The word type list contains the word type names and the corresponding ID which we have used in the dictionary. The word type IDs 1-36 is same as in Apache OpenNLP. Any value above that is used to define unique behaviors when translating.

How to navigate through the system?

In all pages you can find the navigation menu. From the navigation menu you can navigate to the Main Page, Add Definition/Rule page and Download Dictionary page.
